## 核心设计

### 语义理解分类 (v2.0)

```
┌─────────────────────────────────────────────────────────┐
│                    AI 编码助手                          │
├─────────────────────────────────────────────────────────┤
│  1. 获取知识库结构（lark_api.py --get-structure）        │
│     - 支持完整层级结构（最多 20 层）                     │
│     - 智能标识 [可用分类] 节点                           │
│                                                          │
│  2. 读取文档内容（Read 工具）                             │
│                                                          │
│  3. AI 语义分析：                                      │
│     - 主题类型（技术教程/产品文档/行业报告...）           │
│     - 内容性质（原创/转载/工作文档...）                   │
│     - 目标受众（开发者/产品经理/普通用户...）             │
│                                                          │
│  4. 语义匹配最佳分类目录                                  │
│     - 主题匹配（技术类 → 技术目录）                      │
│     - 用途匹配（教程 → 学习/笔记）                       │
│     - 受众匹配（内部文档 → 相关部门）                    │
│                                                          │
│  5. 调用 lark-cli 执行上传                               │
│     - lark-cli docs +create \                           │
│       --title "标题" \                                  │
│       --markdown "内容" \                               │
│       --wiki-node <父节点 token>                         │
└─────────────────────────────────────────────────────────┘
```

## 模块说明

### lark_api.py - API 封装层

| 函数 | 功能 |
|------|------|
| `check_lark_cli_installed()` | 检查 lark-cli 是否安装 |
| `load_config()` / `save_config()` | 配置文件管理 |
| `extract_space_id_from_url()` | 从 URL 提取 space_id |
| `get_node_info()` | 获取节点信息 |
| `create_document()` | 创建文档 |
| `move_node()` | 移动节点 |
| `update_node_title()` | 更新节点标题 |
| `delete_node()` | 删除节点 |
| `build_node_tree()` | 构建完整节点树（支持 20 层） |

### uploader.py - 智能上传器

| 功能 | 说明 |
|------|------|
| 单文件上传 | 上传单个 Markdown 文件 |
| 批量上传 | 上传整个目录 |
| 自动分类 | 基于 AI 语义理解 |
| API 限流 | 自动延迟 1000ms |
| 进度显示 | [1/5] 格式进度提示 |

### analyzer.py - 结构分析器

| 检测项 | 说明 |
|--------|------|
| 层级过深 | 超过 4 层的目录结构 |
| 空分类 | 没有子节点的文件夹 |
| 孤立节点 | 根目录下的文档 |
| 命名不一致 | 中英文空格、特殊字符等 |

### formatter.py - 文档格式化器

| 格式化项 | 规则 |
|---------|------|
| 空行统一 | 最多 2 个连续空行 |
| 行尾空格 | 移除所有行尾空格 |
| 标题标准化 | `#` 后添加空格 |
| 列表修复 | `-` 后添加空格 |
| 链接标准化 | 统一链接格式 |
| 大文件跳过 | > 100KB 时跳过复杂处理 |

### optimizer.py - 优化执行器

| 操作 | v2.0 状态 |
|------|----------|
| 删除空分类 | ✅ 可执行 |
| 重命名节点 | ✅ 可执行（自动生成规范名称） |
| 移动节点 | ⚠️ 需要指定目标位置 |
