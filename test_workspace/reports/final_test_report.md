## Bug 修复记录

### Bug #1: 列表格式化错误 ✓ 已修复
- **问题**: 列表格式化将 `-List` 处理成 `-L`，只保留首字母
- **原因**: 正则表达式 `r'^(\s*[-*+])([^\s])'` 只匹配一个字符
- **修复**: 改为 `r'^(\s*[-*+])(\S.*)'` 匹配整个列表项
- **验证**: UF-004 测试通过

### Bug #2: 缺少导入错误 ✓ 已修复
- **问题**: formatter.py:264 调用 `require_config()` 但未导入
- **修复**: 添加 `from lark_api import require_config`
- **验证**: II-008 测试通过
