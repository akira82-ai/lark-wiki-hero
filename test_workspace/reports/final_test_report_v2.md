1. **Bug #1**: 列表格式化错误

   - 修复: 正则表达式从 `([^\s])` 改为 `(\S.*)`

2. **Bug #2**: 缺少导入错误
   - 位置: `formatter.py:9`
   - 修复: 添加 `from lark_api import require_config`

1. **--get-structure 命令**
