## 发现的问题

### Bug #1: 列表格式化错误 (严重)
**位置**: `formatter.py` - 列表格式修复逻辑
**问题**: 将 `-List` 处理成 `-L`，只保留首字母
**影响**: 用户使用格式化功能时，列表项内容丢失
**复现**: 使用 `--fix-lists` 规则格式化包含列表的文件
**建议**: 检查列表格式修复的正则表达式逻辑

### Bug #2: 缺少导入错误 (严重)
**位置**: `formatter.py:264`
**问题**: 调用 `require_config()` 但未导入
**影响**: 直接运行 formatter.py 会崩溃
**修复**: 添加 `from lark_api import require_config`
```python
# 在 formatter.py 第8行后添加
from lark_api import require_config
```
