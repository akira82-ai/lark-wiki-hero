# 飞书知识库节点数据字段说明

## 当前已获取的字段（fetch_all_nodes_live）

| 字段 | 类型 | 说明 | 用途 |
|------|------|------|------|
| token | string | 节点唯一标识 | 唯一识别、引用 |
| title | string | 节点标题 | 显示、搜索 |
| obj_type | string | 对象类型 (docx, bitable, sheet, file, wiki) | 内容类型分析 |
| obj_edit_time | string | 对象编辑时间戳 | 更新频率分析 |
| has_child | boolean | 是否有子节点 | 结构分析 |
| parent_node_token | string | 父节点 token | 层级关系 |
| depth | int | 层级深度 | 深度分析 |
| path | string | 完整路径 (通过标题拼接) | 路径展示、分类 |

## API 可返回但未获取的字段

| 字段 | 类型 | 说明 | 潜在用途 |
|------|------|------|----------|
| creator | string | 创建者 ID | 贡献者统计 |
| node_create_time | string | 节点创建时间 | 创建趋势分析 |
| node_type | string | 节点类型 (origin, mirror) | 节点性质分析 |
| obj_create_time | string | 对象创建时间 | 内容年龄分析 |
| obj_token | string | 对象 token | 获取详细信息 |

| origin_space_id | string | 原始空间 ID | 跨空间分析 |

| space_id | string | 空间 ID | 多空间管理 |

## 可额外获取的信息（需要二次 API 调用）

### 1. 文档详细信息
```bash
GET /open-apis/docx/v1/documents/{obj_token}
```
可获取：
- 文档内容长度（字符数）
- 段落数量
- 标题层级结构
- 图片、表格数量
- 附件信息

### 2. 文件元信息
```bash
GET /open-api/drive/v1/files/{obj_token}
```
可获取：
- 文件大小（bytes）
- 文件类型（扩展名）
- 上传时间
- 下载链接

### 3. 评论和互动
```bash
GET /open-apis/docx/v1/documents/{obj_token}/comments
```
可获取：
- 评论数量
- 讨论活跃度
- 协作质量

### 4. 访问统计

```bash
# 需要通过其他 API 或日志系统
```
可获取：
- 浏览次数
- 访问用户数
- 热门内容排行

## 潜在的深度分析能力

### 1. 时间维度分析
- **内容新鲜度**: 基于 obj_edit_time
- **创建趋势**: 基于 node_create_time
- **更新频率**: 编辑时间间隔统计
- **活跃度**: 近期更新内容占比

### 2. 协作维度分析
- **贡献者分布**: 基于 creator/owner
- **协作质量**: 评论数量、互动频率
- **所有者分布**: 内容归属统计
- **共建程度**: 多人协作内容占比

### 3. 内容质量分析
- **文档长度**: 字符数统计
- **结构完整度**: 标题层级、段落结构
- **媒体丰富度**: 图片、表格、附件数量
- **可读性**: 文本长度、段落密度

### 4. 使用热度分析
- **访问频率**: 浏览次数（需要额外数据源）
- **搜索命中**: 搜索结果出现次数
- **链接引用**: 被其他文档引用次数
- **分享次数**: 分享到外部次数

### 5. 内容主题分析
- **关键词提取**: 从标题提取高频词
- **标签推断**: 基于路径和标题推断标签
- **主题聚类**: 内容相似度聚类
- **知识图谱**: 节点关联关系
